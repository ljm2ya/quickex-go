오케이 이제 난 새 거래소를 추가할거야 거래소 이름은 phemex 고
  api 문서는 https://phemex-docs.github.io/#overview 이거 다 읽어보면돼.
  우선 spot보다는 futures 위주로 기능구현할거고
  현재 전체 프로젝트의 구조에 맞게끔 짜야해.
  설명을 하자면 core 에 interface.go가 있어
  여기서 SubscribeQuotes 은 실시간으로 websocket을 구독해서 가격을 
  불러오는거고
  FetchBalance 에서 futuresPosition 이 true 면 선물 포지션의 총 가격 
  불러오면돼.
  왜 구분해놨나면 spot과 다르게 선물은 레버릿지를 쓰니깐 계정잔고가 아닌 
  포지션 전체의 잔고를 불러와야해.

  interface.go 에 있는 모든 함수들은 하나도빠짐없이 모든기능을 반드시 
  구현해야하고,
  rest를 절대 쓰지않고 websocket으로 작동하게끔 구현해야돼. (거래 포함)

  api 키는 내가 특정위치에 파일로 저장할테니깐 그거 불러오는건 나중에 나한테 
  물어보고,
  웹소켓부분은 api문서를 잘 살펴봐야해. 핑퐁이나 하트비트 이런것들과 api limit
   즉 웹소켓 연결 횟수나 연결가능한 심볼 한도 예외들 전부 살펴서 잘 되게끔 
  처리해야돼

  최종적으로는 모든 기능에 대한 테스트를 하고 모든기능의 동작이 잘 되는지 
  확인이되야 종료야
